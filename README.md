# Prueba para Proyecto AIS 

## Historia de Usuario
```
El usuario mediante su identificacion (cedula/ruc) al iniciar debe crear un 
usuario para poder hacer uso de todas las funcionalidades de la aplicación,
posteriormente puede elegir los colores que desea utilizar tanto en el 
background como el font y guardar sus cambios.
Asimismo el usuario puede crear a su personal y los puestos disponibles en el 
panel Registro de Información y en el panel de Relación tablas puede relacionar 
estos nodos a su gusto, todos sus cambios se quedan guardados desde que crea al
usuario y puede observarlos en Home donde aparece una tabla de logs.

```
### Paquetes importantes utilizados
```
La presente aplicación utiliza buenas practicas como es el uso de renv, 
gargoyle, klmr/box, R6 según las especificaciones de la prueba. 

```
